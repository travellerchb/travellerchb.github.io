Markdown + Typora 笔记速记

发表于 2024-08-02 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 2 分钟

一、标题 （# + 空格 / CTRL + 0 ~6 / CTRL + 加号/减号）

# 一级标题
## 二级标题
### 三级标题

#### 四级标题
##### 五级标题
###### 六级标题

快捷键

CTRL + 1~6 ：选中文本变为对应级别的标题
CTRL + 0 ：选中标题变为普通文本
CTRL + 加号/减号 ：选中标题进行级别加减

标题会在目录与大纲分级显示, 可以跳转

严格模式开启：标题为# + 空格，而不是#

二、段落

1. 换行(回车 / SHIFT + 回车)

回车 ：大换行（分隔大）
SHIFT + 回车 ：小换行（分隔小)

2. 分割线（--- 或*** + 回车）

三、字体

1. 粗体（** ** 或 __ __ 包围 / CTRL + B）

2. 下划线（<b> </b> 包围 / CTRL + U）

3. 斜体（* * 或 _ _ 包围 / CTRL + I）

4. 删除线（~~ ~~ 包围 / SHIFT + ALT + 5)

5. 高亮（== == 包围）

6. 斜体并强调 （用 *** *** 或 ___ ___ 包围）

四、上下标

1. 上标（^ ^ 包围）

2. 下标（~ ~ 包围）

五、区块（> + 空格）

最外层

第二层

第三层

六、列表

1. 无序列表（- / + / * + 空格 【回车 + Tab 会进入下级】）

  • 1
    • 2
      • 3
        • ~~

2. 有序列表（数字 + . + 空格 【回车 + Tab 会进入下级】）

  1. 1
    1. 2
      1. 3
        1. ~~

3. 任务列表（ - [ ] + 空格 / - [x] + 空格）

七、代码显示

1. 单行代码（` ` 包围 / CTRL + SHIFT + `）

2. 代码块（``` + 回车 / CTRL + SHIFT + K）

八、链接（直接写链接 / [] + () / CTRL + K 【可用于文内跳转 [~~](#~~)】)

九、注释（文中写[^ + 数字] 文后加上[^ + 对应数字]:）

十、表格（CTRL + T / 用 | 绘制）

1 2 3
1 2 3
4 5 6

十一、图片（![]() / 直接拖拽 / 复制粘贴）

十二、表情（打开编辑，找到表情与符号）

十三、目录（[TOC]）

十四、其余快捷键

按键 效果 按键 效果
Ctrl + D 选中当前词 Ctrl + L 选中当前句/行
Ctrl + E 选中当前区块 Ctrl + F 搜索当前选中
Ctrl + H 替换当前选中 Ctrl + J 滚动屏幕将选中滚至顶部
Ctrl + W 关闭当前窗口 Ctrl + N 打开新窗口
Ctrl + O 打开文件 Ctrl + P 搜索文件并打开
Ctrl + 回车 表格下方插入行 Ctrl + , 打开偏好设置
Ctrl + . 切换全角/半角标点 Ctrl + / 切换正常/源代码视图
Ctrl + Shift + - 缩小视图缩放 Ctrl + Shift + + 放大视图缩放
